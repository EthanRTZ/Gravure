# Gravure du Lion — Site vitrine

Mini site statique trois pages (accueil, projets, contact) pour présenter la micro-entreprise de gravure artisanale Gravure du Lion.

## Structure

- `index.html` — présentation générale, services, workflow et galerie photo.
- `projects.html` — sélection de petites séries illustrées et filtrables.
- `contact.html` — formulaire de brief, coordonnées et horaires sur rendez-vous.
- `assets/css/style.css` — style global minimaliste et responsive.
- `assets/js/main.js` — interactions légères (menu mobile, formulaire, filtres, animations).

## Utilisation

Ouvrir `index.html` dans un navigateur moderne ou servir le dossier via un serveur statique (ex. `python -m http.server`).
